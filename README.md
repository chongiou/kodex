# Kodex
Kodex（拉丁语意为 "书册"，发音 `/ˈkoʊ.deks/` ，同 "Codex" ） 是一个为自动精灵平台设计的声明式UI框架

# 实现
- [x] 值更改事件 : onChange
- [x] 弹窗已创建Hook : dialogCreated
- [ ] select 元素更改默认选中 : 本身不提供,但可以通过更改 options 数组来实现
